SECTION 8: TROUBLESHOOTING GUIDE
═══════════════════════════════════════════════════════════════════════════════

Issue: Port 8008 already in use
Solution: 
  - Kill existing process: netstat -ano | find "8008"
  - Change port in config.yaml (api_port: 8009)
  - Use alternative port in startup command

Issue: Virtual environment not activating
Solution:
  - Run: venv\Scripts\activate.bat
  - Verify prompt shows (venv)
  - Use full path to python: venv\Scripts\python.exe

Issue: Import errors for langchain modules
Solution:
  - Run: pip install -r requirements.txt
  - Use virtual environment python
  - Clear pip cache: pip cache purge

Issue: Vector store not loading
Solution:
  - Check vectorstore/faiss_index/ exists
  - Verify index.faiss and index.pkl present
  - Rebuild index if necessary

Issue: Rate limiting too strict
Solution:
  - Edit config.yaml (rate_limit_enabled)
  - Adjust limits in main.py decorators
  - Increase per-IP limits if needed
